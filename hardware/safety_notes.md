# Safety Design

UV-C radiation is harmful to human skin and eyes.

The following safety mechanisms are implemented:

1. Door Interlock
A limit switch ensures the UV lamp only activates when the chamber door is fully closed.

2. Relay Isolation
The relay separates the high-voltage UV lamp circuit from the low-voltage control electronics.

3. Fixed Exposure Duration
The timer ensures the lamp operates only for the required sterilization duration.

4. Enclosed Chamber
The reflective chamber prevents UV leakage outside the system.
